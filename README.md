# API Notes Markdown

Ce projet est une API Node.js développée avec Express.
Elle permet d’exposer des notes de cours rédigées en Markdown sous forme de routes HTTP dynamiques.

Chaque fichier `.md` ajouté dans le dossier `notes/` devient automatiquement accessible via l’API.

---

## Objectif pédagogique

Ce projet sert de base à :

- L’apprentissage du versioning avec Git et GitHub
- La mise en place d’un workflow collaboratif
- L’introduction future à GitHub Actions et au CI/CD

---

## Prérequis

- Node.js version LTS
- Git
- Visual Studio Code

Vérifier l’installation de Node :

```bash
node -v
```
