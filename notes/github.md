# GitHub : Le réseau social des développeurs

GitHub, c'est un peu le réseau social des développeurs, mais avec des outils de pointe pour construire des logiciels. C’est la plateforme incontournable pour stocker, gérer et partager du code source.

## Le cœur : Git

GitHub repose sur Git, un système de "versioning" (gestion de versions). Cela permet de garder une trace de chaque modification apportée à un projet. Si vous faites une erreur, vous pouvez revenir en arrière en un clic.

## Le Cloud pour le code

Au lieu de garder vos fichiers sur votre ordinateur, vous les hébergez sur des "repositories" (dépôts) en ligne. Cela permet de :

Collaborer : Plusieurs personnes peuvent travailler sur le même projet simultanément sans écraser le travail des autres.

Sauvegarder : Votre code est sécurisé et accessible partout.

## L'aspect Communautaire (Open Source)

C'est la plus grande bibliothèque de code au monde. Vous pouvez :

- Consulter le code de projets célèbres (comme VS Code ou React).

- Proposer des améliorations (Pull Requests).

- Signaler des bugs (Issues).
